# proton
A programming language that is written from scratch.

## Filetype
Default source file extension: ``.pfc``.
Default header file extension: ``.pfh``.

## Syntax

### Importing header files
You can import header files with **eat** keyword. nom nom

source.pfc:
```c
eat "header.pfh"
```
